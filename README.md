About the Project

ReWear** is a full-stack Spring Boot web application that encourages sustainable fashion by allowing users to **Buy, Sell, Donate, and Exchange** clothes. The platform helps reduce textile waste while giving clothes a second life.

# Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/ReWear.git


## Open Project

Import the project into Eclipse or IntelliJ IDEA.

---

## Configure Database

Create a PostgreSQL database.

Example:

```
Database Name : user
```

Update **application.properties**

```properties
spring.datasource.url=jdbc:postgresql://localhost:5432/rewear
spring.datasource.username=postgres
spring.datasource.password=your_password

spring.jpa.hibernate.ddl-auto=update
spring.jpa.show-sql=true
```

---

## Run Project

Run

```
SmartWasteClothesCollectionApplication.java
```

Open browser

```
http://localhost:8080
```
